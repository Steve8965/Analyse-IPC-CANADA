# Analyse de l’IPC au Canada

## Présentation
Ce projet analyse l’évolution de l’Indice des prix à la consommation (IPC) au Canada entre 2015 et 2025, afin d’identifier les principales composantes de l’inflation et de proposer une prédiction pour 2026.

## Objectifs
- Analyser l’évolution de l’IPC
- Identifier les composantes clés de l’inflation
- Étudier les relations entre variables
- Construire un modèle de régression
- Réaliser une prédiction simple

## Données
Source : Statistique Canada (https://www.statcan.gc.ca/fr/debut)  
Période : 2015 – 2025  

## Méthodologie
- Analyse descriptive
- Visualisation
- Corrélation
- Régression
- Prédiction

## Résultats
- Hausse continue de l’IPC
- Accélération après 2021
- Importance du logement et de l’alimentation
- Multicolinéarité entre variables
- Prédiction IPC 2026 ≈ 168,98
